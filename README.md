# School Project


### Запускаем проект локально
- клонируем репозиторий:
- `git clone https://github.com/baigashl/school`


- заходим в проект:
- `cd school`


- запуск с помощью docker:
- `docker-compose up -d --build`
- 
- после:
- `docker-compose run web python manage.py migrate`



- переходим по ссылке:
- http://0.0.0.0:8000/
